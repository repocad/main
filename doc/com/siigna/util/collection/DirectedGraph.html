com.siigna.util.collection

DirectedGraph

case class DirectedGraph [V, A] (_arrows: Iterable[((V, A), V)]) extends Iterable[((V, A), V)] with Product with Serializable

An immutable directed graph with named arrows. An arrow has a type of ((V, A), V) and can be written as V -> A -> V.

Example code for instantiating a graph: 

  val g = DirectedGraph( 'a -> 1 -> 'b,
                         'a -> 2 -> 'a,
                         'b -> 1 -> 'a )       // Using multiple arguments.

  val g = DirectedGraph(List( 'a -> 1 -> 'b,
                              'a -> 2 -> 'a,
                              'b -> 1 -> 'a )) // You can use any Collection.
You can also use new DirectedGraph(...) if you prefer that.

_arrows

the arrows that defines this graph.

  21. def apply (path: (V, A)): Option[V]

    Walks the graph, going from a vertex through an arrow to a new vertex.

    Walks the graph, going from a vertex through an arrow to a new vertex. The result is the vertex you reach.

    Here is an example graph (only one arrow): 

           1
  A -------> B
    By specifying A -> 1 you will reach B.

    Example code: 

      val g = DirectedGraph( 'a -> 1 -> 'b,
                         'a -> 2 -> 'a,
                         'b -> 1 -> 'a )
  var vertex = 'a
  vertex = g(vertex -> 2) getOrElse('x) // vertex is still 'a.
  vertex = g(vertex -> 1) getOrElse('x) // vertex is now 'b.
  vertex = g(vertex -> 2) getOrElse('x) // vertex is 'x (edge not found).

    You can use pattern matching to determine whether you reached a vertex: 

      g('a -> 1) match {
    case Some(vertex) => println("Reached vertex: " + vertex)
    case None => println("Path not found.")
  }

    path

    a vertex and arrow to walk through, ie. 'a -> 1.

    returns

    Some(vertex) if you reach a new vertex, None if path not found.
