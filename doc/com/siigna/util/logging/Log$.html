  27. var level : Int

    The debug-level.

    The debug-level. Levels are following:

    • Error - Critical errors, preventing code from being executed.
    • Warning - Unexpected behaviour causing the program to choose alternate actions.
    • Info - Information about code being passed.
    • Debug - Debug-information. Note: Can log quite a bit of lines.
    • Success - Reports when something succeeds... Sometimes.
