com.siigna.app.model

DynamicModel

trait DynamicModel extends Thread

The DynamicModel contains the dynamic (var dynamic : Seq[DynamicShape]) layer which is the part of the drawing which are selected out for editing - the mutable and globally accessible canvas. The DynamicModel is also responsible for propagating actions to the underlying Model. These shapes can be altered without changes in the static layer. When the changes have been made, the shapes are removed from the dynamic layer, and the actions which have been applied on the dynamic layer is applied on the static layer.
